## 第一部分：长流程链路（L1~L4）

### L1: 数据全生命周期闭环 ★★★
```
传感器数据 → L1自动审核 → L2智能审核 → L3人工复核 → 告警触发 → 告警确认 → 转工单 → 工单处理 → 办结
```
| # | 步骤 | 操作 | 预期结果 | 实际 |
|---|---|---|---|---|
| 1.1 | 登录 | admin/admin123 | 跳转驾驶舱 | |
| 1.2 | 数据审核Tab | 预警中心 → 数据审核 | 看到统计卡片（待审/L1/L2/L3/归档数） | |
| 1.3 | 触发自动审核 | 等待后台10min调度或手动触发 | L1-L2完成后状态更新 | |
| 1.4 | 确认识别异常 | 列表中看到suspicious/rejected记录 | 状态正确、auto_reason含编码 | |
| 1.5 | L3人工复核 | 勾选smart_reviewed记录→核准/驳回 | 操作后状态变为archived/manual_reviewed | |
| 1.6 | 告警列表 | 回到告警列表 | 看到对应级别的告警 | |
| 1.7 | 确认告警 | 点确认按钮 | 状态变为acknowledged | |
| 1.8 | 转工单 | 点转工单 | 工单创建成功，告警关联工单号 | |
| 1.9 | 工单处理 | 工单管理 → 处理工单 → 提交办结 | 工单→closed，告警自动→resolved | |
